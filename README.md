# 🌐 Portfolio Website

A modern and fully responsive personal portfolio built using Next.js, TypeScript, ReactBits UI, and Tailwind CSS.

---

## 🚀 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- ReactBits UI
- Vercel (Deployment)

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI
- Fast performance
- Reusable components

---

## 🧠 Challenges Faced

### Layout Management
I faced issues while implementing different layouts instead of relying on the default app layout in Next.js. Managing multiple layouts while keeping consistency across pages was challenging.

### Responsiveness
Making the website fully responsive took time. Adjusting components and layouts for different screen sizes required careful work.
